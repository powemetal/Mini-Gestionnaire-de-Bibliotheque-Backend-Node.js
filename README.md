# Mini-Gestionnaire-de-Bibliotheque-Backend-Node.js
Ce projet est un mini-backend développé en TypeScript avec Node.js, Prisma ORM et une base de données PostgreSQL hébergée sur Neon. L'objectif principal de cet atelier est de concevoir et de manipuler la couche d'accès aux données (CRUD) ainsi que de gérer des relations entre entités sans utiliser de framework Web
